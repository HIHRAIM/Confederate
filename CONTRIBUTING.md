## Versioning Guidelines

Please follow this versioning format:

**MAJOR.FEATURE.MINOR**

- **MAJOR** — global or breaking changes that significantly affect the project.
- **FEATURE** — introduction of a new feature or substantial functionality.
- **MINOR** — small improvements, fixes, or non-breaking adjustments.

Localization fixes are considered **minor** changes.

If you are unsure how to version your contribution, consult other repository editors before submitting changes.
